# 🎯 Guess The Number Game

A modern and interactive number guessing game built using HTML, CSS, and JavaScript.

The program generates a random number between 1 and 100, and the user tries to guess it with hints provided after every attempt.

---

# 🚀 Features

- Random number generation
- Interactive gameplay
- Too High / Too Low hints
- Attempt counter
- Best score tracking
- Dark mode support
- Responsive design
- Smooth animations
- Restart game feature

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

# 🎮 Game Rules

- The computer selects a random number between 1 and 100.
- Enter your guess.
- The game gives hints:
  - Too High
  - Too Low
- Continue until you guess correctly.
